# EcceMe AI Coding Instructions
This is a Hugo static site using the Blowfish theme and deployed through Cloudflare Pages from GitHub.
## Working rules
- Do not modify the theme source directly unless explicitly asked.
- Prefer Hugo overrides in layouts/, assets/, content/, and config/.
- Make one focused change at a time.
- Explain which files will change before editing.
- Keep the site mobile-first.
- Preserve the dark minimal style.
- Avoid heavy JavaScript unless there is a clear reason.
- Do not delete content without asking.
- Do not commit changes automatically.
## Site direction
The site should feel personal, grounded, and useful, not stiff or corporate.
The homepage should eventually include:
- Hero intro
- Featured projects
- Coursework progress
- Course retrospectives
- Recent writing
- About section
- Contact / GitHub / resume links
## Writing tone
Direct, plainspoken, reflective, and specific.
Avoid fake professionalism, startup hype, motivational filler, and vague phrases like "passionate multidisciplinary"
