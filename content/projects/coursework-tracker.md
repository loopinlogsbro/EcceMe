## Problem

Coursework progress is easy to lose track of when the work moves quickly. A visible tracker makes the current state obvious.

## What I built

The homepage now has a progress section. The next step is filling it with the official completed, current, and remaining course list.

## What I learned

Progress data should be accurate or explicitly marked as provisional. Publishing guessed numbers would make the site less useful.

## Current status

The structure exists. The official degree-plan data still needs to be entered.

## Next steps

Add completed courses, current course, remaining courses, percent complete, and target finish window from the WGU plan.
